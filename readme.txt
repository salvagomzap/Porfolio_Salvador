# Portfolio Profesional - Salvador Gomez

Este es mi portfolio digital profesional, desarrollado como parte de mi formación en desarrollo de aplicaciones web y administración de bases de datos. El sitio está diseñado para ser completamente responsive y destacar mis competencias técnicas junto con mis mejores proyectos.

## 🚀 Tecnologías Utilizadas
* **HTML5 & CSS3** (Estilos personalizados y transiciones analíticas)
* **Bootstrap 5** (Grid System, componentes de UI y maquetación responsive)
* **FontAwesome** (Iconografía técnica)

## 📁 Estructura del Proyecto
* `/css` - Estilos finos y efectos hover personalizados.
* `/js` - Lógica de interacción.
* `/assets` - Recursos globales (imágenes y documentación).
